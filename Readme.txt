== Description ==
WpTutsBD Admin Panel plugin is for Styling Your Own WordPress Dashboard Panel and its absolutely free .It allows users to easily customize the look of the administration interface for WordPress . Now with this plugin you can re-brand your Websites back-end too.

Key Features: 

* Nice n Clean design 
* 5 color schemes: Blue (default), Green, Red, Yellow, Purple and  (Custom color/CSS panel) 
* Custom login Page 
* Custom logo or Text 
* Choose your Menu icons 
* Easy installation 
* Multilanguage support  ( .PO .MO included ) 

Get more information and support at [the official Wptutsbd Admin panel page](http://wptutsbd.com/custom_dashboard_plugin).


== Installation ==

1. Upload the `/Wptutsbd_admin` folder to your `/wp-content/plugins/` directory.
2. Activate the plugin through the `Plugins` menu in WordPress.
3. Navigate to `Dashboard->WPtutsBD Admin` and Start Customizing.

Get more information and support at [the official Wptutsbd Admin panel page](http://wptutsbd.com/custom_dashboard_plugin)
